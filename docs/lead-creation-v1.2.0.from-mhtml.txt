Case HTTP Status Code Body Description Created 201 LeadCreationResult JSON Object The Lead was accepted and entered into our system. Parameter Error 400 ErrorResponse JSON Object Illegal Method 405 Method not allowed Conflict 409 ErrorResponse JSON Object The lead will not be accepted because the lead already exists or for business reasons (e.g. not feasible). The supplied data was well understood and sufficient but didn't meet the business requirements. LeadCreationResult JSON Object

Field Type Required Comment leadStage string yes The stage of the lead depending on the data provided.

qualification If the lead was accepted but contact info or technical feasibility could not be checked (e.g. because of missing fields)
discovery if the lead was accepted and is considered feasible. However data is missing to create an offer automatically
selling if the lead was accepted and contained all required information in order to create an offer. dataAcquisitionLink String Only if leadStage ��� selling A link that takes the lead to an online form where they can submit the missing information from DA1-3. So they can advance to the selling stage.

If the lead is already in stage selling the link will be empty. appointmentBookingLink String Only if leadStage == selling A link that takes a lead where they can book an appointment with Vamo.

If the lead is not in stage selling the link will be empty. ErrorResponse JSON Object
